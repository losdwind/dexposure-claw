\subsection{DeXposure-FM Architecture}
\label{sec:dexposure-fm-architecture}

DeXposure-FM leverages a pre-trained graph foundation model (GraphPFN~\cite{eremeev2025g2tfm}) to represent the inter-protocol credit-exposure network, augmented with lightweight task-specific heads for downstream prediction. The model supports three objectives: (1) multi-step forecasting of edge existence and weights; (2) evaluation of model robustness during historical stress events; and (3) imputation of missing edge information.

\subsubsection{Inputs and notation (aligned with the DeXposure construction)}
Let $t\in\mathcal{T}$ denote discrete observation times and let $\tau = t_2-t_1$ be a discrete interval between consecutive times.
For each interval $\tau$ we construct a weighted directed graph snapshot
\begin{equation}
G_{\tau}=(\mathcal{P}_{\tau},\mathcal{E}_{\tau}),
\end{equation}
where $\mathcal{P}_{\tau}$ is the set of protocols and $\mathcal{E}_{\tau}$ is the set of directed exposure links.
Node weights are given by $w_{\tau}(p)$ (the protocol's end-of-interval stock) as in Eq.~\eqref{eq:node-weight}, and edge weights are given by $w_{\tau}(e_{pq})$ (the change in credit exposure from $p$ to issuing protocol $q$ over $\tau$) as in Eq.~\eqref{eq:edge-weight}, where token-level value flows $F^{\sigma}_{pq,\tau}$ are defined by Eq.~\eqref{eq:value-flow}. Credit exposure exists when $X_p(t)\cap X_G(q)\neq\varnothing$ (Eq.~\eqref{eq:credit-exposure-definition}).

For each protocol $p$ we construct node features that combine structural and categorical information:
\begin{itemize}
  \item \textbf{Structural features:} log-transformed TVL $\log(1+w_{\tau}(p))$, token count, maximum token share, and composition entropy.
  \item \textbf{Categorical features:} one-hot encoded protocol category (e.g., DEX, Lending, Bridge, Staking).
  \item \textbf{Graph structure:} $G_{\tau}$ with edge set $\mathcal{E}_{\tau}$ representing credit exposures between protocols.
\end{itemize}

\subsubsection{Graph encoder}
We use GraphPFN~\cite{eremeev2025g2tfm}, a pre-trained graph foundation model, as the backbone encoder. GraphPFN extends a tabular foundation model (LimiX) with graph-aware attention layers that incorporate neighborhood structure. Given node features $\mathbf{x}_p$ and graph $G_{\tau}$, the encoder produces $d$-dimensional node embeddings:
\begin{equation}
\{\mathbf{h}_{p,\tau}\}_{p\in\mathcal{P}_{\tau}} = E_{\mathrm{GraphPFN}}\!\left(G_{\tau}, \{\mathbf{x}_{p}\}_{p\in\mathcal{P}_{\tau}}\right).
\end{equation}

We evaluate two training regimes:
\begin{itemize}
  \item \textbf{Frozen encoder:} The pre-trained GraphPFN weights are frozen; only task heads are trained (linear probing).
  \item \textbf{Fine-tuned encoder:} All parameters are updated end-to-end with a unified learning rate.
\end{itemize}

\subsubsection{Task heads}
\paragraph{(1) Link prediction head.}
For edge existence prediction, we use a Hadamard-product decoder followed by an MLP:
\begin{equation}
\hat{a}_{\tau+h}(p,q) = \sigma\!\left(\mathrm{MLP}(\mathbf{h}_{p,\tau} \odot \mathbf{h}_{q,\tau})\right),
\end{equation}
where $\sigma$ is the sigmoid function. The same decoder outputs edge weight predictions $\widehat{w}_{\tau+h}(e_{pq})$ via a separate linear layer.

\paragraph{(2) Node prediction head.}
For node-level forecasting (e.g., future TVL), we apply a simple MLP to node embeddings:
\begin{equation}
\widehat{w}_{\tau+h}(p) = \mathrm{MLP}_{\mathrm{node}}(\mathbf{h}_{p,\tau}).
\end{equation}

\paragraph{(3) Evaluation tasks.}
We evaluate DeXposure-FM on three complementary tasks:
\begin{itemize}
  \item \textbf{Multi-step forecasting:} Predict edge existence and weights at horizons $h \in \{1, 3, 7\}$ weeks.
  \item \textbf{Shock analysis:} Evaluate model performance during historical stress events (Terra/Luna collapse, FTX collapse) to assess robustness under distribution shift.
  \item \textbf{Imputation:} Reconstruct randomly masked edges to evaluate the model's ability to infer missing exposure information.
\end{itemize}
Network-level statistics (Gini coefficient, HHI, density) are computed deterministically from predicted graphs for analysis.
