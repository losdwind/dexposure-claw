\paragraph{Reproducibility.}
A complete leaderboard run finishes in under four GPU-hours on a
single RTX~4090 plus approximately \$10 of LLM API spend, making the
harness accessible without industrial compute. The harness fixes
seeds, snapshots library versions, and hashes checkpoints; every
reported number reproduces bit-exactly from the released
artefacts.
