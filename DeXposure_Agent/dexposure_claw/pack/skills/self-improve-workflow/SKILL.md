# Self-Improve Workflow

Use this when repeated failures suggest the agent should learn a new procedure.

Allowed automatic proposals:

- Better troubleshooting instructions.
- Clearer report templates.
- Safer install snippets.
- More precise benchmark run checklists.

Requires human approval:

- Benchmark metric definitions.
- Ground-truth construction.
- Test split changes.
- Leaderboard protocol changes.
- LLM decision prompt changes.

Never silently alter reported benchmark scores or frozen evaluation definitions.
