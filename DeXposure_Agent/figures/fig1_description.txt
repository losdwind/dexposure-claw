Figure 1: DeXposure-Agent System Overview

DeXposure-Agent is an automated weekly risk monitoring system for decentralized finance (DeFi) credit exposure networks. It takes as input a time-varying directed graph G_t, where nodes represent DeFi protocols (e.g., Aave, MakerDAO, Curve) and weighted edges represent credit exposures between them. The system runs a five-stage pipeline. First, a Data Health Gate validates snapshot quality. Then, a graph foundation model (DeXposure-FM, built on GraphPFN) forecasts the future network structure G_{t+h} at horizons of 1-12 weeks via Monte Carlo sampling. A Risk Monitor computes seven structural metrics (concentration, connectivity, PageRank inequality, etc.) and flags anomalies using z-scores against historical baselines. A Stress Testing module simulates five adversarial scenarios -- top-protocol failure, bridge cluster collapse, stablecoin de-peg, lending shock, and correlated defaults -- quantifying contagion losses via graph propagation. Finally, a Decision Engine (powered by an LLM agent) synthesizes alerts and stress results into actionable recommendation tickets with severity ratings, target protocols, supporting evidence, and natural-language rationale. The key research claim is that combining the FM backbone with LLM reasoning (FM+LLM) produces better risk decisions than either FM+rules or pure LLM approaches alone.
